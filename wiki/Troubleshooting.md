# Troubleshooting & Fixes

## Common Problems & Solutions

### Add-on fails to install
- Make sure you are using the correct zip structure: the zip should contain a folder named `plugin.video.nlziet` with all files inside.
- Use the latest release from GitHub.

### Streams not playing / DRM errors
- Enable `inputstream.adaptive` in Kodi settings.
- Check your NLZiet subscription status.
- Restart Kodi after changing settings.

### Login issues
- Double-check your NLZiet credentials.
- If 2FA is enabled, ensure you complete the process in the browser if prompted.

### Add-on not showing up after install
- Restart Kodi.
- Check Kodi logs for errors.

## Getting Help
- Open an issue on [GitHub](https://github.com/Nigel1992/NLZiet-Kodi-Addon/issues) with your Kodi version, device, and a debug log.
