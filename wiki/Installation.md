# Installation Guide

## Prerequisites
- Kodi 19 or newer
- NLZiet subscription
- Widevine DRM support (for encrypted streams)

## Step-by-Step Installation
1. Download the latest release zip from the [GitHub Releases page](https://github.com/Nigel1992/NLZiet-Kodi-Addon/releases).
2. Open Kodi and go to **Add-ons > Install from zip file**.
3. Select the downloaded `plugin.video.nlziet.zip` file.
4. Wait for the installation notification.
5. Go to **Add-ons > My Add-ons > Video Add-ons > NLZiet** and open the add-on.
6. Enter your NLZiet credentials in the add-on settings.
7. (Optional) Enable `inputstream.adaptive` in Kodi for DRM playback.

## Updating
- To update, repeat the installation steps with the latest zip.

## Uninstalling
- Go to **Add-ons > My Add-ons > Video Add-ons > NLZiet > Uninstall**.
